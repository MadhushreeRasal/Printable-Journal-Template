## Printable Journal Template

A beginner-friendly HTML and CSS project created while learning about CSS borders, gradients, layouts, and styling.
This webpage is designed as a printable daily journal template with sections for tasks, thoughts, moods, achievements, and sketches.

## Live Demo

"https://madhushreerasal.github.io/Printable-Journal-Template/"

## Technologies Used
HTML5
CSS3
Google Fonts

## Features
Daily log section
To-do list
Mood tracker
Achievement section
Thought / brain dump space
Doodle area
Decorative floral theme
Multiple border styles and gradients

## What I Practiced

This project was mainly created to practice:

CSS borders and border styles
Gradients and background styling
Floats and positioning
Basic webpage layout design
Typography using Google Fonts
Creating themed static webpages

## Learning Experience

While building this project, I experimented with different border types, colorful layouts, and decorative styling to better understand how CSS properties affect webpage design.
